TypeFacet Autokern Installation

Back to the TypeFacet Autokern Manual.

Step 1: Download TypeFacet

  • Download as .zip
  • Download as .tar
Once downloaded, expand the archive.

Step 2: Download the dependencies

  • Python 2.7 or later. TypeFacet Autokern may work with earlier versions of Python.
  • robofab
  • pystache (only if using --log-dst option to write html logs)
  • svgwrite (only if using --log-dst option to write html logs)
It is not necessary to install these packages into your Python installation. Instead, create a "dependencies" folder in your "typefacet" folder and move these packages there.

Step 3: Test Installation

  • Go to your Typefacet folder in Console (Mac) or Command Prompt (Windows)
# Mac
cd ~/Downloads/typefacet

# Windows
cd Downloads/typefacet
  • Invoke Autokern With Usage Switch
# We need to temporarily append our dependencies to the Python path.

# Mac:
env PYTHONPATH=dependencies/robofab:dependencies/pystache:dependencies/svgwrite:python/src-main python python/src-main/autokern/Autokern.py -h

# Windows:
set PYTHONPATH=%PYTHONPATH%;dependencies/robofab;dependencies/pystache;dependencies/svgwrite
python python/src-main/autokern/Autokern.py -h
Using the -h help switch (as above) should output the usage message: 
usage: Autokern.py [-h] --ufo-src UFO_SRC --ufo-dst UFO_DST
                   [--log-dst LOG_DST]
                   [--glyph-pairs-to-kern GLYPH_PAIRS_TO_KERN [GLYPH_PAIRS_TO_KERN ...]]
                   [--glyphs-to-kern GLYPHS_TO_KERN [GLYPHS_TO_KERN ...]]
                   [--min-distance-ems MIN_DISTANCE_EMS]
                   [--max-distance-ems MAX_DISTANCE_EMS] [--assess-only]
                   [--do-not-modify-side-bearings]
                   [--allow-negative-side-bearings] [--precision PRECISION]
                   [--intrusion-tolerance INTRUSION_TOLERANCE]
                   [--intrusion-limit-glyph-width-fraction INTRUSION_LIMIT_GLYPH_WIDTH_FRACTION]
                   [--min-non-intrusion-ems MIN_NON_INTRUSION_EMS]
                   [--kerning-threshold-ems KERNING_THRESHOLD_EMS]

etc.
